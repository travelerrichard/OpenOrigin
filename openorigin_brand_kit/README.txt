OpenOrigin Brand Kit
====================

Contents:
- PNG variants (full color, monochrome, inverted, square icon)
- SVG vector variants (scalable for print and digital)
- Favicon set: 16x16, 32x32, 64x64
- README badge (SVG)

Brand Guidelines:
-----------------
Primary Colors:
- Navy: #0A1F44
- Green: #2EA44F
- White: #FFFFFF
- Black: #000000

Typography:
- Font: Sans-serif, recommended 'Inter' or 'Open Sans'
- Logo text weight: Bold for 'Open', Semi-bold for 'Origin'

Usage:
------
- Use PNG for web contexts where vector is not required.
- Use SVG for high-resolution print or infinite scaling.
- Always maintain clear space around logo equal to the height of the "O".
- Minimum size for digital use: 24px height.

Favicon Use:
- 16x16 for browser tabs.
- 32x32 for desktop shortcuts.
- 64x64 for high-DPI displays.

Verification Badge:
- The included 'OpenOrigin Verified' badge can be embedded in GitHub README.md:
